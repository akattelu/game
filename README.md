This is a collection of graphics examples I'm building using sokol and Zig.

So far there are 3 examples:
- **GLTF Viewer:** Loads and renders a GLTF model via GLB files. Assets are loaded from: https://github.khronos.org/glTF-Assets/
- **Terrain Generation (CPU):** Calculates noise on the CPU and uses it to create and render dynamic terrain.
- **Terrain Generation (GPU):** Calculates noise on the GPU and uses it to create and render dynamic terrain. Comes with an animation feature.
